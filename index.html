Applet Inventory

niggly.py NM
Window Focus Rules (IF/THEN)
When you focus one window, specified others auto-minimise. IFTTT-style card UI for defining rules. Lives in the system tray as NM.
tiles.py TL
Window Tiles + Desktop Canvas
All active windows rendered as tiny coloured tiles, grouped by category. Two modes: compact sidebar list, or full-screen desktop canvas with free-placement and drawable zones. Passthrough ghost mode drops to 10% opacity for click-through. System tray icon TL.
launcher.py MI
Master Suite Launcher
Gamification-style command centre. Each applet shown as a glowing orb — launch individually or all at once. Tracks XP, levels, and launch stats. System tray icon MI.
launch_all.pyw
Auto-launcher (Silent)
Silent launcher for all applets. Drop a shortcut into the Windows Startup folder and everything fires on login — no console window, no fuss.

Config Files

File Stores
niggly_config.json IF/THEN focus rules — which windows to auto-minimise when a target window gains focus
tiles_config.json Tile colours, category groupings, sidebar layout preferences, and window filter rules
canvas_config.json Desktop canvas free-placement positions, drawable zone definitions, and ghost-mode opacity settings
launcher_config.json Launch counts per applet, total launches, XP points, and current level

How to Run

Run everything at once (silent, no console):

# Double-click or add a shortcut to shell:startup
pythonw launch_all.pyw

Run individual applets:

# Master Suite Launcher (gamification UI)
python launcher.py

# Window Focus Rules
python niggly.py

# Window Tiles + Desktop Canvas
python tiles.py

Dependencies:

pip install pywin32 pystray Pillow
