# :speaking_head: NACHO

> Voice AI assistant where every sentence is an interactive hyperlink.

**File:** [`nacho.py`](../../nacho.py)
**Version:** 2.0.0
**Tray Icon:** NA

---

## The Problem

AI chat interfaces are walls of text. You get a response but can't do anything with individual sentences. You can't copy just one line, send it somewhere, or act on it without manually selecting text. The output is read-only and inert.

## The Solution

A voice AI assistant where every single sentence is a hyperlink. Hover to underline, click to open an action bar with options: Copy, Email, Telegram, Thread, Claude, CodePen, Note, Linker, Delete, Wrong (retry). Your own messages get Edit, Spell Check, and AI Check buttons. Every piece of text is actionable.

## Features

- :microphone: Voice input via Vosk offline STT
- :loud_sound: Voice output via pyttsx3 TTS
- :brain: Claude API for conversation
- :link: Every sentence is a clickable hyperlink
- :hammer_and_wrench: Action bar: Copy, Email, Telegram, Thread, Claude, CodePen, Note, Linker, Delete, Wrong
- :pencil2: Edit, Spell Check, AI Check on your own messages
- :page_facing_up: Export chat history
- :floppy_disk: Auto-exports on close

## How It Works

Speech input via **Vosk** offline STT. Responses generated by the **Claude API**. Text-to-speech output via **pyttsx3**. The UI renders each sentence as a styled hyperlink in a **tkinter** text widget. Hover triggers underline styling. Click opens a contextual action bar with integration options. Chat auto-exports to file on close.

## Configuration

Uses the shared API key from `kidlin_config.json`:

```json
{
  "api_key": "sk-ant-..."
}
```

## Usage

Launch `nacho.py`. Speak or type to start a conversation. Claude responds with voice and text. Hover over any sentence to see the underline. Click to open the action bar. Choose an action — copy, email, send to Telegram, etc. Edit your own messages with the inline buttons. Chat auto-saves on exit. Tray icon **NA** provides quick access.
